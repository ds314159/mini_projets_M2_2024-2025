Le projet est composé des parties suivantes:

- article_compiled.pdf : Article compilé en PDF.

- source_article : Code source de l'article en Latex.

- code_projet_manifold : Code ayant servi au protocole expérimental.


La partie code a été réalisée sur Colab, en réponse à des contraintes de disponibilité de hardware et certaines librairies (par exemple, le projet Rapids n'est pas disponible pour Windows).

La partie code_projet_manifold est composée des parties suivantes:

- fichier_travail_1 : notebook de génération de données synthétiques et expérimentations dessus.

- fichier_travail_2 : notebook de travail sur Fashion_mnist.

- results, best_params : dossiers de sauvegarde pour les résultats d'applications des différentes méthodes et de certains paramètres après calibrage.


Le notebooks contiennent les sorties de cellules, et les deux ont été rejoué en entier et sauvegardés avant de les envoyer.

Si toutes fois vous souhaitiez les exécuter, une idée simple serait de mettre le dossier dans un drive google et de les ouvrir avec colab, dans l'idéal dans un environnement A100.
