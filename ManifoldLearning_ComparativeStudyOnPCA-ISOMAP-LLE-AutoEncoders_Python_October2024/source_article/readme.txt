This is OUP's new document class file for typesetting articles for the journal Microscopy and Microanalysis,
"mam-authoring-template.cls".  

mam-authoring-template.cls is based upon the oup-authoring-template.cls which in turn is based upon the standard LaTeX document class
article.cls. It uses natbib.sty for bibliographical references.

The file manifest.txt provides a list of the files in the mam-authoring-template
bundle.  The following are the main files available:
- mam-authoring-template.tex, Sample template file to start with a new article
- mam-authoring-template.pdf, Sample pdf
- mam-authoring-template.cls is the class file 

The documentation file is oup-authoring-template-doc.tex in the doc directory.  To
compile it:
latex oup-authoring-template-doc.tex

Also, oup-authoring-template-doc.pdf is available in the doc directory.
